# ViperUsage
B-VIPER Example Using Instagram Unofficial Api

# Basic Configuration
1. Android Studio 3 RC2
2. Java8 enabled
3. MinSDK 15
4. TargetSDK 26
5. Kotlin version added

# Used Library
1. RxJava2 and RxAndroid
2. Retrofit2 and OkHttp3 Interceptor
3. Gson
4. Glide 4.2.0
5. DataBinding (only Java version)
6. Kotlin Extension
